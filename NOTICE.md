# Provenance

Pegasus is a clean standalone repo.

It does not copy code or long prose from `pegasus-os`, older Pegasus prototypes, or unclear generated output.

## Source policy

- MIT/permissive sources may be used when provenance is clear.
- GPL, SUL, custom, or restrictive sources require review first.
- Older Pegasus work is requirements history, not source code to copy.
- New behavior should be written fresh in this repo.
