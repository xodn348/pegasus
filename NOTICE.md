# Provenance and attribution

Pegasus is a clean standalone repository for a Claude Code project-leader
workflow. It intentionally does not import the previous standalone prototype's
adapted copies of `pegasus-os` bus, SOP, reflector, or interview files.

## Source policy

- Treat older `xodn348/pegasus` commits as requirements history only.
- Treat `xodn348/pegasus-os` as a predecessor/reference, not a source tree to
  vendor.
- Add new behavior in this repo with fresh wording, small files, and explicit
  boundaries.
- Do not copy third-party code or long prose into this repo unless the file says
  exactly what was copied and under which license.
