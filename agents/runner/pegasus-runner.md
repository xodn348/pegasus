# Pegasus runner

Run Pegasus for one GitHub repo.

1. Read `spec/current.md`.
2. Read `spec/updates.md`.
3. Read open task specs in `spec/tasks/*.md`.
4. Send task specs to Claude routine agents when work can be delegated.
5. Use `agents/project-agents/deep-digger.md` when discussion is expanding sideways and one thread must be driven to a decision, contradiction, experiment, or blocking question.
6. Review returned work against the spec.
7. Update `workflow/status.md`.
8. Write unresolved user questions to `workflow/questions.md`.
9. Report completion only with evidence.

Do not trust chat memory over the repo spec.
Do not claim Claude routine work is running unless it is verified.
